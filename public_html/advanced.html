Local Storage for Data Persistence

In this project, we use the localStorage API to maintain cart data even after the user refreshes the page or navigates away from it. Here's how it's done:

Implementation Steps

  1. When an item is added to the cart, the cart array is updated, and the array is converted into a JSON string: 
    localStorage.setItem('cart', JSON.stringify(cart));
  2. When the cart page loads, the stored cart data is retrieved from localStorage and parsed back into an array: 
    const savedCart = JSON.parse(localStorage.getItem('cart')) || [];
  3. The application restores the cart’s state, displaying the items and their quantities: 
    // Example function to render the cart
function renderCart() {
    const cart = JSON.parse(localStorage.getItem('cart')) || [];
    cart.forEach(item => {
        // Update UI with item data
    });
}

How it Works

This method allows the application to maintain the cart's state across page refreshes or when the user navigates to other pages. The data is stored in the browser’s local storage, which persists even after the session ends.

jQuery for Modal Content Update

The jQuery library is used to listen for button clicks on the menu page and dynamically update the content of a modal. The modal shows details of the dish when the respective card is clicked.

Implementation Steps

  1. Add a button to trigger the modal: 
    <button class="btn btn-info" data-bs-toggle="modal" data-bs-target="#dishModal" data-title="Dish Name" data-description="Delicious dish details here">More Info</button>
  2. Use jQuery to listen for the modal's `show.bs.modal` event and dynamically update its content: 
    $('#dishModal').on('show.bs.modal', function (event) {
    var button = $(event.relatedTarget); // Button that triggered the modal
    var title = button.data('title');
    var description = button.data('description');
    
    var modal = $(this);
    modal.find('.modal-title').text(title);
    modal.find('.modal-body p').text(description);
});

How it Works

When a user clicks a button, the relevant data (like the dish title and description) is passed to the modal via `data-*` attributes. jQuery listens for the click and updates the modal's content dynamically without needing to reload the page.

Summary

Both features help enhance the user experience by improving data persistence and enabling dynamic content display without refreshing the page. These techniques are common in modern web development and ensure a more seamless interaction for the user.
